# Advocate Kartik Yadav Website

Single Vite + React + Tailwind CSS project for the Advocate Kartik Yadav website.

Live site: https://shrutee2002.github.io/advocate-kartik-yadav-website/

## Run Locally

**Prerequisites:** Node.js 20+

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy

GitHub Actions builds the root project with `npm install --include=optional` and `npm run build`, then deploys only the generated `dist` folder to GitHub Pages.
