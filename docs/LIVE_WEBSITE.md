Developer: Build & Preview
Prerequisites: Node.js 20+ and npm.

Install dependencies:

```sh
npm install
```

Start local dev server:

```sh
npm run dev
```

Build production bundle:

```sh
npm run build
```

Preview production build locally:

```sh
npm run preview
```

Deployment notes
- GitHub Pages should use "GitHub Actions" as the Pages source.
- The workflow at `.github/workflows/deploy.yml` builds the root project and deploys only `dist`.
- Live URL format: `https://YOUR-GITHUB-USERNAME.github.io/advocate-kartik-yadav-website/`
